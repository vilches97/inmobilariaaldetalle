# Inmobiliaria Al Detalle

Sitio web profesional para la inmobiliaria Al Detalle en Martos, Jaén.

## Características

- Catálogo de propiedades inmobiliarias
- Información de contacto y ubicación
- Diseño responsive y moderno
- Optimizado para motores de búsqueda (SEO)

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui components

## Desarrollo

```bash
npm install
npm run dev
```

## Construcción

```bash
npm run build
```
"# inmobilariaaldetalle"
